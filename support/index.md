# Support - Ma Naturalisation Française

Bienvenue sur la page de support de l'application **"Ma Naturalisation Française"**.

## Contact

Pour toute question, problème ou suggestion concernant l'application, vous pouvez nous contacter :

**Email :** monappnaturalisation@gmail.com

## Assistance

### Problèmes fréquents

**L'application ne se charge pas correctement**
- Vérifiez que vous avez une connexion Internet active
- Assurez-vous d'avoir la dernière version de l'application installée
- Redémarrez l'application

**Questions sur les fonctionnalités**
- Contactez-nous par email pour toute question spécifique

## Signaler un bug

Si vous rencontrez un problème technique, merci de nous envoyer un email avec les informations suivantes :
- Description détaillée du problème
- Étapes pour reproduire le problème
- Version de l'application
- Modèle de votre appareil et version du système d'exploitation
- Captures d'écran si possible

## Suggestions d'amélioration

Nous sommes toujours à l'écoute de vos suggestions pour améliorer l'application. N'hésitez pas à nous faire part de vos idées par email.

## Confidentialité

Pour plus d'informations sur la façon dont nous traitons vos données personnelles, consultez notre [Politique de Confidentialité]({{ '/privacy/' | relative_url }}).
